看完了最低赔后，然后看中赔和高赔
欧赔的分析，我倾向于“跷跷板理论”，以平赔为轴，在最低赔不小于2的情况下，如果负赔太高，资金就会流向胜赔，造成胜赔过热，打出的话庄家赔付会很高，一般不会打出，其他情况同理。当最低赔大于1.5小于2时，对应的是亚盘半球和半一，半一的话135体系是主胜正常打出的情况，即对应胜赔在1.7~1.8，平赔在3.x，负赔在5.x，这时候可以理解为主胜降水将赔付，平负高开任博，如果是134就要防下爆冷的可能，这里要说明一下为什么和“跷跷板理论”有点出入，因为负赔开太高了，超出正常区间，会有一些贪心的资金流入超高赔上，资金总体流向还是平衡的，而且低赔打出也有点“食之无味”的感觉。同样地，亚盘为半球时，胜平负开133体系是正常的，如果临场赔到了134，也可以用负赔高开解释，主胜打出。

欧赔开出后，数值高低不等，有必要将其划分为几个区域来识别。按老欧核的大区分法，不论主客，以一赔（让球方）为首，其区间正好可按让球盘口划分，平手起步，平半为二区，半球为三区，半一为四区，半一往上以此类推。

如果不清楚某场球威廉一赔对应哪个区间，可以按照威廉赔率结构表来对号入座。
