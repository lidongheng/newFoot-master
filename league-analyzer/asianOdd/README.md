3、我也说下我分析的思路：
（1）核心：亚盘分析的核心是拉力。哪个队的拉力越强，哪个队就越受热，结果就越不可能打出。
（2）那怎么看出拉力呢？底层基础知识是要清楚球队在联赛中的广实定位，如果有联赛球队的广实定位更好，没有的话，只能认为庄家初始开的盘口是符合广实定位的，通过受注后的变盘不断调整再确定哪个队拉力强，投注它的反面。
在这里，我以25-26赛季英超为例列举英超联赛球队的广实定位：
第一档次：阿森纳、曼城、切尔西、利物浦
第二档次：热刺、曼联、阿斯顿维拉、纽卡斯尔联
第三档次：埃弗顿、布莱顿、水晶宫、伯恩茅斯
第四档次：诺丁汉森林、布伦特福德、富勒姆、桑德兰
第五档次：西汉姆联、利兹联、伯恩利、狼队

分档次主要看盘口开的合理不合理，一般来说，同档次的球队对阵，主场球队是让平半合理，随着赛季的深入，球队的档次可能会发生微小的变化。比如现在是英超第13轮，我给出的不同档次间对阵开出的盘口为：
同档次主场打同档次：主让平半
第一档次主场打第二档次：主让半一
第一档次主场打第三档次：主让一球
第一档次主场打第四档次：主让一球/球半
第一档次主场打第五档次：主让球半/两球
第二档次主场打第三档次：主让平半或半球
第二档次主场打第四档次：主让一球
第二档次主场打第五档次：主让一球/球半或主让球半
第三档次主场打第四档次：主让半球
第三档次主场打第五档次：主让一球
第四档次主场打第五档次：主让平半
球队主客对调的话，就大约降三个盘

同档次球队让平半，主客对调大约降三个盘可以针对所有联赛。但是由于精力有限，大多数球队无法给出广实定位，只能以庄家开的盘当参考。
