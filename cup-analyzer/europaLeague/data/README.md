# europaLeague/data 说明

| 文件 | 说明 |
|------|------|
| `c113.js` | 球探杯赛 **113** 赛程与积分榜等 |
| `l113.js` | 联赛阶段亚盘盘路榜 |
| `bs113.js` | 大小球盘路榜 |
| `td113.js` | **入球时间分布**（若仓库中暂无，可从球探导出 `td113.js` 放入本目录） |
| `冠军赔率.md` | 欧罗巴夺冠赔率，用于广实/档次参考 |
| `球队与序号对照表.md` | 球队中文名 ↔ 球探序号（可由 `c113.js` 的 `arrTeam` 生成） |

**赛季**：`c113.js` 内 `arrCup` 含赛季信息；爬虫更新时使用 `CUP_ANALYZER_CUP=europaLeague` 与 `25-26` 赛季目录。
