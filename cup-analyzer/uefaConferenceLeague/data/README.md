# uefaConferenceLeague/data 说明

| 文件 | 说明 |
|------|------|
| `c2187.js` | 球探杯赛 **2187** 赛程与积分榜等 |
| `l2187.js` | 联赛阶段亚盘盘路榜 |
| `bs2187.js` | 大小球盘路榜 |
| `td2187.js` | **入球时间分布**（若仓库中暂无，可从球探导出 `td2187.js` 放入本目录） |
| `冠军赔率.md` | 欧会杯夺冠赔率，用于广实/档次参考 |
| `球队与序号对照表.md` | 球队中文名 ↔ 球探序号（可由 `c2187.js` 的 `arrTeam` 生成） |

**赛季**：`c2187.js` 内 `arrCup` 含赛季信息；爬虫更新时使用 `CUP_ANALYZER_CUP=uefaConferenceLeague` 与 `25-26` 赛季目录。
