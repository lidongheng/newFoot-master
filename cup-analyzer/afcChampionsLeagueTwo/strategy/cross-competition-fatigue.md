# 跨赛事疲劳与轮换

> 亚冠联2球队通常同时征战 **国内联赛**、可能还有 **国内杯赛** 与 **国家队比赛**（国际比赛日）。

## 1. 赛程密度（模板）

| 球队 | 近 7 天场次 | 亚冠联2下一场 | 联赛下一场 | 间隔天数 | 轮换风险 |
|------|-------------|------------|------------|----------|----------|
| | | | | | 高/中/低 |

## 2. 评估维度

- **体能**：间隔短、旅途远 → 轮换概率高。
- **战意**：联赛争冠/保级压力 vs 亚冠联2淘汰赛阶段 → 可能在某条线留力。
- **阵容厚度**：替补与青训能否支撑双线。

## 3. 与世界杯分析的差异

- 俱乐部 **磨合度** 通常高于国家队，但 **比赛负荷** 更高。
- 无「默契球」式小组末轮，但存在 **轮换** 与 **选择性发力**。
